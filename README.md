# Nest
Nest.js framework utils.

## Development

### Requirements

- [Make](https://www.gnu.org/software/make/)
- [Docker](https://www.docker.com/)
- [Docker Compose V2](https://docs.docker.com/compose/)

### Commands

#### Run tests

```bash
make test
```

#### Run tests on "watch" mode

```bash
make test-watch
```
